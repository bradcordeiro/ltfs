    class LTFS::LTFSExtentInfo

    Poor tape-writing practice causes large variations in Extent information about an LTFSFile, so this part of information has been abstracted to its own class to do some sanity checks.

def initialize(extentinfo)
  @extentinfo = extentinfo
  if @extentinfo.nil? or @extentinfo[:extent].nil?
    handle_missing_extent
  elsif @extentinfo[:extent].is_a?(Array)
    handle_extent_array
  else
    handle_single_values
  end    
end
